## Current Architecture

### Tech Stack
- **Framework**: Astro 6.x (Static Site Generation)
- **Styling**: Custom CSS with CSS variables (dark/light theme support)
- **Search**: Client-side Fuse.js for resource filtering
- **Content**: MDX for tutorials + JSON for resources
- **Deployment**: GitHub Pages via GitHub Actions

## 🚀 Deployment Checklist

Before deploying to GitHub Pages, update `astro.config.mjs`:

1. **Set your GitHub site URL**:
```javascript
site: 'https://yourusername.github.io',  // Replace with your username
base: '/devhub',  // Replace with your repo name (remove if repo is username.github.io)
```

2. **Update GitHub workflow** (optional): Ensure `.github/workflows/deploy.yml` has correct Node version

3. **Push to GitHub**:
```bash
git add .
git commit -m "Initial deployment"
git push origin main
```

4. **Enable GitHub Pages**: Go to Settings → Pages → Select "GitHub Actions"

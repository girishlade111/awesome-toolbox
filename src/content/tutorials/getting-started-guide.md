## Introduction

Welcome to DevHub! This guide will walk you through the basics of using our community-driven platform to discover and contribute developer tools.

## What is DevHub?

DevHub is a curated collection of:

- **Custom Agents** - AI-powered tools for specific development tasks
- **Prompts** - Optimized prompts for AI models
- **Instructions** - Step-by-step guides for common workflows
- **Plugins** - Extensions that enhance your development tool

## Browsing Resources

1. Navigate to the **Browse** page from the navigation menu
2. Use the search bar to find specific tools
3. Filter by category using the category buttons
4. Click on tags to find resources with similar functionality
5. Sort resources by date, name, or popularity

## Finding What You Need

When looking for resources:

- Start with the search bar and enter keywords related to your need
- Use category filters to narrow down to specific types
- Click on tags that interest you to see related resources
- Check the Learning Hub for guided tutorials

## Making Your First Contribution

Ready to give back? Here's how:

1. Prepare your resource (agent, prompt, instruction, or plugin)
2. Go to the **Submit** page
3. Follow the contribution guidelines
4. Open a pull request on GitHub
